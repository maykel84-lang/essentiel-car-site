## Guide de saisie WooCommerce — boutique.essentielcar.com

**Comment utiliser ce document :**
- Champ **Nom** → copier le "Titre produit"
- Champ **Description courte** → visible AU-DESSUS du bouton Ajouter au panier
- Champ **Description** (onglet) → coller en mode Texte/HTML
- **Yoast / RankMath** → Meta title + Meta description + Focus keyphrase
- **Permalink** → vérifier le slug

> ⚠️ Les champs marqués "À confirmer sur fiche fournisseur CJ" doivent être complétés avec les données réelles avant publication.
