**Comment utiliser ce document :**
- Champ **Nom** → copier le "Titre produit"
- Champ **Description courte** → visible AU-DESSUS du bouton Ajouter au panier

- **Yoast / RankMath** → Meta title + Meta description + Focus keyphrase
- **Permalink** → vérifier le slug
